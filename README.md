# WeatherApp
> Demo : https://ryudg.github.io/WeatherApp/

## Description
- [OpenWeatherMap API](https://openweathermap.org/) 를 활용한 날씨 표현
- GeoLocation 사용 현재 위치의 날씨 표현
- 국내 지역 자동 검색 기능 구현
